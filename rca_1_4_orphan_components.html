2. Root Cause Analysis

⚠ Exploratory Development & Leftover Scaffolding
During initial UI exploration and template evaluation, multiple component libraries and landing page variations were prototyped. When the final design language was selected and wired into page.tsx routes, the unselected prototypes and demo files were left behind in the active component directories rather than being cleaned up or archived.
  1. No Garbage Collection Workflow: The team lacked a periodic pruning process or automated dead-code detection during pull request reviews.
  2. Co-location of Demos: Third-party UI library examples (*-demo.tsx) were dumped directly into src/components/ui/ alongside core production primitives.

3. Business & Technical Impact

  • Cognitive Overhead: Developers and code maintainers waste time investigating, refactoring, or testing components that are not even part of the live website.
  • Refactoring Hazard: When global refactorings occur (such as Server Component migrations), orphaned components trigger TypeScript errors or build failures, dragging down velocity.
  • IDE & Search Clutter: Searching for symbol references or component names yields false positives from abandoned demo code.

4. Remediation Plan

We propose a clean structural remediation:

  1. Option A (Recommended) — Move & Clean:
    • Create an archive directory: src/_archive/components/.
    • Move all 13 orphaned section components and Sidebar.tsx into src/_archive/ so exploratory work is preserved without cluttering production folders.
    • Delete the 8 *-demo.tsx files from src/components/ui/.
    • Document active vs. archived components in README.md.
  2. Option B — Complete Deletion: Delete all 21 orphaned files directly, relying on Git history if prototype code needs to be recovered later.

5. Verification Results

  1. Dead Code Scan: Verified that zero imports in active route pages reference any of these 21 files.
  2. Build Safety: Removing or archiving these files will have zero impact on the live website or production bundle.
